### 制約
- N <= 8

bitを用いた解法

- N <= 40
半分全列挙
O(2 ^ (N / 2))
- N <= 300
O(N ^ 3)
- N <= 5000  
O(N ^ 2)
- N <= 200000  
O(N log N) or O(N)
- N <= 1000000000  
O(1) or O(log N) or O(sqrt(N))
- N <= 1000000000000000000  
O(1) or O(log N)
